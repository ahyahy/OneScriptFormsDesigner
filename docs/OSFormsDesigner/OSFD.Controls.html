Элементы управления дизайнера

Окно дизайнера можно условно разделить на три области:

- Панель компонентов в левой части окна
- Панель вкладок с проектируемыми формами посредине окна
- Панель свойств в правой части окна

Панели сбоку имеют разделитель, которым можно изменить её размер.

Панель компонентов

В панели компонентов можно с помощью мыши выбрать нужный компонент и поместить его на форму.

Выберите компонент, наведите курсор мыши на контейнер, в который вы хотите его добавить, и щелкните левой кнопкой мыши. Появится элемент управления с координатами сетки мыши в качестве верхних / левых координат элемента управления. Элемент "Указатель" не является инструментом и служит для отмены взятия компонента из списка.

Список компонентов может быть расширен с выходом новой версии дизайнера.

Размер панели можно для удобства уменьшить до минимума. Панель нельзя закрыть.

Панель вкладок

В середине расположена панель вкладок, на вкладках которой можно одновременно работать со множеством разных проектируемых форм. Это удобно. Мы можем на одной вкладке открыть главную форму приложения, на другой диалоговое окно, открываемое из главной формы и т.д.

В ярлычке вкладки выводится иконка дискеты, имя вкладки с именем находящейся на ней формы, заключенным в скобки, и крестик для закрытия вкладки. Иконка дискеты окрашивается красным при наличии не сохраненных изменений на форме. Вы можете закрывать вкладки, но оставшуюся последнюю незакрытую вкладку удалять не разрешается.

Панель свойств

Панель свойств служит для просмотра и редактирования значения свойств элементов управления. Над ней расположено поле выбора для выбора уже размещенных на форме элементов управления.

Ещё выше находится дерево объектов с возможностью сортировки объектов по имени или по порядку добавления на форму.

Размер панели так же можно уменьшить до минимума но нельзя закрыть.

Главное меню

Главное меню включает в себя меню Файл, Правка, Вид, Инструменты, Помощь.

Через меню Вид подменю Сценарий можно просмотреть код ещё не сохраненного будущего сценария, то есть не из кода модели а по состоянию в представлении модели в дизайнере. Через меню Инструменты подменю Запуск можно запустить на исполнение форму так же по состоянию в представлении модели в дизайнере.

Для подменю Запуск назначены горячие клавиши Ctrl+F6. Для Windows 10 действие сработает только с левой клавишей Ctrl.
