Справка библиотеки форм

Для удобства рекомендую держать под рукой открытую справку по библиотеке форм (Справка по формам). В ней есть подробные примеры, иерархия пространства имен. Можно будет копировать целые куски кода и вставлять в свой сценарий.

В разделе Дополнения есть дополнительные разъяснения и две игры написанные с применением библиотеки форм. В скачанном релизе форм будет каталог со справкой на случай отсутствия подключения к интернету. Содержимое скачанной справки соответствует версии библиотеки форм.
