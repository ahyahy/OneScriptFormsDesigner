ДизайнерФормДляОдноСкрипта (OneScriptFormsDesigner) Конструктор

Синтаксис

Дизайнер(<Параметр>)

Параметры

Параметр (необязательный)
Тип: Строка.
Состояние окна консоли.

Описание

Параметр может принимать одно из двух строковых значений: ВосстановитьКонсоль и СкрытьКонсоль.

Примечание

Пример

ДФ.Дизайнер("ВосстановитьКонсоль"); // окно консоли развёрнуто
ДФ.Дизайнер("СкрытьКонсоль"); // окно консоли скрыто
ДФ.Дизайнер(); // окно консоли минимизировано
