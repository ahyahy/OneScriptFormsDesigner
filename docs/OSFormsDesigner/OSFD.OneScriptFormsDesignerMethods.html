ДизайнерФормДляОдноСкрипта (OneScriptFormsDesigner) Методы

Методы класса ДизайнерФормДляОдноСкрипта (OneScriptFormsDesigner) перечислены ниже. Полный список членов класса ДизайнерФормДляОдноСкрипта (OneScriptFormsDesigner) смотрите в разделе ДизайнерФормДляОдноСкрипта (OneScriptFormsDesigner) Члены.

Методы

ДобавитьКонтрол (AddControl) Добавляет элемент управления.
ЗакрытьДизайнер (CloseDesigner) Закрывает дизайнер.
ОткрытьФорму (LoadForm) Открывает форму из файла osd.
ПолучитьЗначениеСвойства (GetPropertyValue) Возвращает значение свойства.
ПолучитьТипСвойства (GetPropertyType) Возвращает тип свойства приведенный к строке.
СохранитьФорму (UnloadForm) Сохраняет форму в файл osd.
СформироватьСценарий (GenerateScript) Сохраняет сценарий os по указанному пути.
УдалитьКонтролы (RemoveControls) Удаляет все элементы управления с формы.
УстановитьСвойство (SetProperty) Устанавливает значение свойства для контрола.
