Установка и запуск

Рекомендую ознакомиться со статьей на ресурсе infostart.ru - Как решать повседневные задачи с помощью OneScript

Скачайте и установите 1Script - язык сценариев для специалистов по 1С

Дизайнер запускается следующим кодом:

ПодключитьВнешнююКомпоненту("ВашКаталогНаДиске\OneScriptFormsDesigner.dll");
ДФ = Новый ДизайнерФормДляОдноСкрипта();
ДФ.Дизайнер();

Сохраните этот код в файл с расширением .os в кодировке UTF-8.

Запуск можно произвести в трёх режимах. С отображением окна консоли, минимизацией или скрытием окна консоли. Укажите для нужного варианта параметр запуска в текстовом формате.

ДФ.Дизайнер("ВосстановитьКонсоль"); // окно консоли развёрнуто
ДФ.Дизайнер("СкрытьКонсоль"); // окно консоли скрыто
ДФ.Дизайнер(); // окно консоли минимизировано
