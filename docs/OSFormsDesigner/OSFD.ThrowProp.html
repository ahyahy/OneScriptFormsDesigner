Установка, изменение, сброс значений свойств

Некоторые особенности работы со свойствами объектов.

Форма

У формы есть свойство Путь. Для формы, которая была сохранена или была открыта из файла osd или os это свойство заполнено. Если свойство заполнено, сохранение формы при вызове пункта подменю Сохранить форму меню Файл произойдет по этому пути. Свойство Путь доступно только для чтения.

Изображения

При сохранении формы или сценария заданные в свойствах компонентов изображения копируются в каталог выбранного файла osd или os. Поэтому удобнее будет сохранять форму или сценарий в специально созданном для этого каталоге.

Для удаления назначенного элементу фонового изображения или изображения кликните дважды по значению поля ФоновоеИзображение или Изображение. Текущее значение при этом будет выделено. Затем нажмите клавишу DEL.

Шрифт

Для возврата к шрифту по умолчанию кликните дважды по значению поля Шрифт. Текущее значение шрифта при этом будет выделено. Затем нажмите клавишу BACKSPACE, затем клавишу ENTER.

Строки

Для ввода новой строки при редактировании свойств с типом значения Строка нажмите ENTER. Для завершения редактирования нажмите CTRL совместно с ENTER. В некоторых случаях сработает только левый CTRL совместно с ENTER.

ПанельСтрокиСостояния

Для вновь созданной панели строки состояния свойство ПоказатьПанели установлено в значение Ложь, поэтому созданные на ней панели изначально не будут отображены.

СеткаСвойств

В режиме дизайнера сетка свойств будет показывать русифицированные свойства объекта, заданного в свойстве ВыбранныйОбъект. В режиме исполнения свойства будут показаны на английском. В разработке.

СеткаДанных

Компонент СеткаДанных имеет для настройки минимум свойств, которые позволяют создать его в скрипте, разместить, задать стили, но не заполнить данными. Дополнить свойства и методы нужно самостоятельно.

Подсказка

Компонент Подсказка работает в дизайнере аналогично невизуальным компонентам. Он будет отображаться ниже формы в панели для невизуальных компонентов.

Вы можете создать несколько подсказок с разными текстами.

У тех элементов управления, которые уже размещенных на форме и которые могут содержать подсказку в палитре свойств появятся свойства ToolTip на Подсказка1, ToolTip на Подсказка2 и т.д.

Свойств будет добавлено соответственно количеству созданных подсказок. Именно там укажите текст для каждой подсказки. Подсказка со строкой не содержащей текста не будет назначаться элементам управления.
