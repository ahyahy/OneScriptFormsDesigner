Дизайнер форм
для сценарного языка OneScript





Первая версия дизайнера вышла 23 февраля 2022 г. с номером 1.0.0.0

Версия 1.3.2.0 - 

Описание
- Исправлена ошибка для объекта ПолеВыбора (ComboBox) при восстановления из сохраненного файла сценария

Версия 1.3.0.0 - 

Описание
- Исправлена ошибка совместимости с предыдущими версиями при отсутствии в сохраненном файле свойства формы ИмяОбъектаФормыДляОдноСкрипта
- Добавлен класс - КольцевойИндикатор (CircularProgressBar)

Версия 1.2.2.0 - 

Описание
- Исправлен алгоритм выгрузки компонента Подсказка. Подсказка со строкой не содержащей текста не будет назначаться элементам управления
Новые свойства и методы дизайнера:
- добавлено свойство - ВизуальныеСтилиФормВключены (FormVisualStylesTurnOn)
- добавлено свойство - ДизайнерОткрыт (DesignerOpen)
- добавлен метод - ДобавитьКонтрол (AddControl)
- добавлен метод - ЗакрытьДизайнер (CloseDesigner)
- добавлен метод - ОткрытьФорму (LoadForm)
- добавлен метод - ПолучитьЗначениеСвойства (GetPropertyValue)
- добавлен метод - ПолучитьТипСвойства (GetPropertyType)
- добавлен метод - СохранитьФорму (UnloadForm)
- добавлен метод - СформироватьСценарий (GenerateScript)
- добавлен метод - УдалитьКонтролы (RemoveControls)
- добавлен метод - УстановитьСвойство (SetProperty)
- Изменения в тексте справки

Версия 1.2.1.0 - 

Описание
- Добавлен вывод на экран сообщения об ошибке при неудачном запуске скрипта на исполнение из дизайнера

Версия 1.2.0.0 - 

Описание
- Добавлена возможность сохранения разрабатываемой формы в сценарий и открытия из сохраненного сценария
- Добавлено свойство для формы СтильСкрипта
- Добавлено свойство для формы ИмяОбъектаФормыДляОдноСкрипта

Версия 1.1.0.0 - 

Описание
- Исправлено некорректное сохранение-восстановление не целых числовых значений для класса ЭлементСписка (ListItem)
- Добавлены методы для автоматического тестирования дизайнера
- Исправлена ошибка при восстановлении сохраненной формы для свойства Выравнивание (Alignment) класса СтильКолонкиПолеВыбора (DataGridComboBoxColumnStyle)
- Добавлен класс - МаскаПоляВвода (MaskedTextBox)
- Добавлен класс - Таблица (DataGridView)

Версия 1.0.2.0 - 

Описание
- Исправлено некорректное восстановление формы из файла osd компонентов, свойства которых имеют значение перечисления СтильПоляВыбора (ComboBoxStyle)

Версия 1.0.1.0 - 

Описание
- Исправлена потеря пробелов для некоторых свойств с типом значения Строка при открытии сохраненной формы из файла osd
